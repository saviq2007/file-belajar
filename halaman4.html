Pengenalan C++

C++ adalah bahasa pemrograman yang dikembangkan sebagai perpanjangan dari bahasa C. Bahasa ini sering digunakan untuk pengembangan sistem, game, aplikasi desktop, dan banyak lagi. C++ mendukung pemrograman prosedural dan berorientasi objek.

Struktur Dasar Program C++


#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

Penjelasan:

  • #include <iostream>: Direktif pra-pemrosesan untuk menyertakan file header iostream yang digunakan untuk input dan output.
  • int main(): Fungsi utama dari program C++ di mana eksekusi dimulai.
  • std::cout << "Hello, World!" << std::endl;: Mencetak "Hello, World!" ke layar.
  • return 0;: Mengakhiri fungsi utama dan mengembalikan nilai 0.

Tipe Data dalam C++

  • int: Tipe data untuk bilangan bulat.
  • float: Tipe data untuk bilangan desimal dengan presisi tunggal.
  • double: Tipe data untuk bilangan desimal dengan presisi ganda.
  • char: Tipe data untuk karakter tunggal.
  • bool: Tipe data untuk nilai boolean (true/false).
  • string: Tipe data untuk rangkaian karakter (diperlukan menyertakan #include <string>).

Contoh Program Sederhana

Program Penjumlahan Dua Bilangan


#include <iostream>

int main() {
    int a, b, sum;
    std::cout << "Masukkan dua bilangan: ";
    std::cin >> a >> b;
    sum = a + b;
    std::cout << "Hasil penjumlahan: " << sum << std::endl;
    return 0;
}

Penjelasan:

  • std::cin >> a >> b;: Menerima input dua bilangan dari pengguna.
  • sum = a + b;: Menjumlahkan dua bilangan yang dimasukkan.
  • std::cout << "Hasil penjumlahan: " << sum << std::endl;: Mencetak hasil penjumlahan ke layar.
